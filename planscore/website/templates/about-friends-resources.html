OpenElections seeks to create the first free, comprehensive, standardized, linked set of election data for the United States, including federal and statewide offices.

Election Geodata is Nathaniel Vaughn Kelso’s collection of precinct shapes for U.S. elections past, present, and future.

Project Linework is Daniel Huffman’s library of handcrafted vector linework for cartography, each designed in a unique aesthetic style.
